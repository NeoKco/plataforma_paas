# Finance Module Source Material

Esta carpeta conserva los documentos fuente usados para definir y ejecutar la construcción del módulo `finance`.

Su propósito es dejar trazabilidad para desarrollo futuro:

- roadmap detallado original del módulo
- prompt maestro de ejecución
- desglose operativo de tareas

Archivos incluidos:

- `roadmap_modulo_finance_alzex_codex_detallado.md`
- `finance_codex_prompt_maestro.md`
- `finance_codex_tareas_ejecucion.md`

Estos documentos se movieron desde la carpeta manual `modulo finanzas/` a `docs/` para que sigan disponibles como referencia histórica del desarrollo del módulo.
