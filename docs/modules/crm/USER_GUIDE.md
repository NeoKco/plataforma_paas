# CRM User Guide

Guía operativa del módulo `crm` (`CRM comercial`) para usuarios tenant y soporte funcional.

## Qué resuelve hoy

`crm` sirve para:

- ordenar oportunidades comerciales
- registrar seguimiento, notas y adjuntos
- mantener histórico de cierres
- armar plantillas reutilizables
- emitir cotizaciones estructuradas

No administra el catálogo de productos.  
El catálogo reusable vive en `products`.

## Vistas principales

- `Resumen`
  lectura rápida del pipeline, histórico y cotizaciones
- `Oportunidades`
  trabajo activo comercial
- `Histórico`
  cierres `won/lost`
- `Cotizaciones`
  propuestas por cliente/u oportunidad
- `Plantillas`
  estructuras reutilizables para cotizar

## Flujo recomendado

1. crear o revisar la oportunidad
2. registrar notas, actividades o adjuntos
3. preparar o elegir una plantilla
4. cotizar usando líneas libres o productos del módulo `products`
5. cerrar la oportunidad cuando corresponda

## Relación con otros módulos

- `business-core`
  clientes, organizaciones y contexto base
- `products`
  catálogo reusable para líneas de cotización y plantillas

## Límites actuales

- no genera PDF formal
- no hace workflow comercial de aprobación
- no reemplaza al catálogo de productos
